# 🛠️ GharCart Admin Panel – Dashboard

The GharCart Admin Panel is a web-based dashboard built using React.js that allows administrators to manage products, users, and orders efficiently in the GharCart e-commerce system.

## 🌐 Live Demo
🔗 Admin Panel (Vercel): https://ghar-cart-admin.vercel.app/

🔗 Backend API: https://gharcart-backend-production.up.railway.app/

## 🚀 Features

📦 Product Management

➕ Add new products

✏️ Edit existing products

❌ Delete products

📋 View all products

👥 User Management

View registered users

Manage user roles (Admin/User)

📦 Order Management

View all orders

Update order status

🔐 Authentication

Secure admin login

Protected routes (only admin access)

🧰 Tech Stack

Frontend

React.js

CSS Tailwind

Axios

Backend Integration

REST APIs from Node.js & Express server

JWT Authentication
